13支整理牌的程式

原本是大一c++課程上的期末作業，原本作業內容是:
寫出一個輸入1~52隨機亂數，長度為13的陣列，
運用自己的邏輯用程式碼去排列，再輸出一串排列好的陣列出去的程式。

老師將每個人的程式碼2個一組來對戰，看哪一方能以同樣的手牌，排出比對方強的牌形。
最後以勝率作為期末成績的依據打分數。

原始的版本只有輸入輸出的功能，沒有寫入顯示的程式碼。
所以我重構了一份新的，所以這個程式誕生了。

規則 : 

隨機發配一個長度13內容是1~52的整數亂數
1~13為梅花1~13，14~26為菱形1~13 ...以此類推

回傳的陣列
arry[0]~arry[2] 為前墩
arry[3]~arry[7] 為中墩
arry[8]~arry[12] 為後墩
